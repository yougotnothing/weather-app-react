# WEATHER REACT APP

This is react app what show you a wether in a choosen city.

## How to run the app

---

#### "instal dependencies"

~~~
yarn / npm i
~~~

---

~~~
yarn dev / npm dev
~~~

---

#### In console you can see the localHost, copy that and paste it in your browser and finally you can run the weather react app!

---

## Enjoy!
